About

Portfolio-ready phishing detection, fully local.

PhishGuard ML is a Flask web app that classifies URLs with a Scikit-learn Random Forest model trained from a local CSV dataset. It does not call any third-party phishing detection API, which makes it suitable for offline demonstrations and privacy-conscious deployments.

Local-first

The model artifact is loaded from model.pkl when Flask starts, and every prediction is generated on the server from extracted URL features.

Explainable signals

Each scan shows the URL length, symbols, HTTPS status, IP-address usage, subdomain count, suspicious keyword count, slashes, and query-parameter presence.

Deployable

The project uses common Python dependencies, environment-variable friendly Flask conventions, and a file layout that is straightforward to deploy on Render.
